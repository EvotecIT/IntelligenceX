About IntelligenceX GitHub App

We use the IntelligenceX GitHub App to simplify setup:

  • ✓ Sign in without creating tokens manually
  • ✓ Pre-fill settings when creating your own GitHub App

Dependabot note: Dependabot PR workflows usually cannot access repository secrets, so review comments may appear as github-actions instead of your app bot.

We don't store your credentials or access repos without your approval. Everything runs locally.
