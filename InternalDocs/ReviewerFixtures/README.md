# Reviewer Fixtures

This folder stores reviewer fixture artifacts used by the `IntelligenceX.Tests` harness.

Current files:
- `review-summary-golden.md`: golden snapshot for reviewer summary formatting tests.
- `review-smoke-inline.cs`: inline review smoke sample.
- `review-suggestion-smoke.cs`: suggestion review smoke sample.

These are intentionally kept under `InternalDocs` so they stay internal and out of website docs.
